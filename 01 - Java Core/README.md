# JAVA Core

Java is a Object Oriented Programming Language developed By Sun Microsystems earlier known as Oak later in 1995 by it's Inventor Jame Gosling it renamed as java.

## Working of java 

Java Code first Compiled Into ByteCode than this ByteCode interpreted into machine Code 

```
java Source code -> Bytecode (Compiled java code) -> MachineCode (Interpreted ByteCode)
```
## Naming Conventions 

*PascalConvention* is used for classes , Interfaces while *camelCase* used for methods and variables.
